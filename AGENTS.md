# Project Instructions

- Keep transcription behavior under `src/features/transcription/`.
- Keep routed composition under `src/views/` and app routing under `src/app/`.
- Keep generic UI and utilities in `src/components/ui/` and `src/lib/`.
- Do not add authentication, PromptPal backend, chat, generation, or user-account dependencies.
- Use red, green, refactor TDD for non-trivial behavior changes.
- Run tests, typecheck, build, formatting, and boundary checks before committing.
- Keep source media and transcript history browser-local.
